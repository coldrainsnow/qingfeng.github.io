想不想让你 Windows 自带的各种动画都变得缓慢？这可别有一番风味啊！顺便还能观察下系统的动画是怎么实现的呢。

---

修改注册表即可实现：

```
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Windows\DWM]
"AnimationsShiftKey"=dword:00000001
```
